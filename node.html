Setting Up a Node.js Server with the http Module?

How It Works

  1. Initial Setup: Show how to initialize a Node.js project (npm init)and create the main server file.
  2. Importing http: Demonstrate requiring the built-in http module.
  3. Creating the Server: Use http.createServer()to define the server logic.
    • Explain the callback function that receives the request(req) and response(res)
  4. Listening for Requests:Use server.listen(port, hostname, callback)to start the server.
  5. Basic Response: Show how to send a simple "Hello World" response using res.writeHead() (for headers/status) and res.end() (for the body).
